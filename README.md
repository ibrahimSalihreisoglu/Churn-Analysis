# Churn Analysis

Customer churn analysis & prediction using Python.

## Overview
This project covers an end-to-end churn workflow:
- Data preparation and basic cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering (age groups, subscription groups)
- Segment-based churn analysis (city, package, gender, reason code)
- Statistical tests (Mann–Whitney U, Welch t-test, Chi-square + Cramér’s V)
- Modeling and comparison (Logistic Regression, Decision Tree, Random Forest, XGBoost)
- Evaluation metrics (Accuracy, Precision, Recall, F1, ROC-AUC)
- Streamlit UI for churn probability prediction

## Tech Stack
- Python, pandas, numpy
- scikit-learn, XGBoost
- matplotlib, seaborn
- scipy, statsmodels
- Streamlit

## Project Structure (suggested)
- `app.py` : Streamlit app (UI + plots)
- `notebooks/` : optional EDA/modeling notebooks
- `src/` : helper functions (optional)
- `requirements.txt` : dependencies
