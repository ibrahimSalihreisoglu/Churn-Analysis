# Churn Analysis

Customer churn analysis & prediction using Python.  
Scope: Data prep → EDA → Feature engineering → Segmentation → Statistical tests → ML modeling/comparison → Streamlit prediction form.

## Tech Stack
- Python, pandas, numpy
- scikit-learn, XGBoost
- matplotlib, seaborn
- scipy, statsmodels
- Streamlit

## Project Content
- EDA & data cleaning (missing values, basic checks)
- Feature engineering (age groups, subscription groups)
- Segment-based churn analysis (city, package, gender, reason code)
- Statistical tests (Mann–Whitney U, Welch t-test, Chi-square + Cramér’s V)
- Modeling: Logistic Regression, Decision Tree, Random Forest, XGBoost
- Metrics: Accuracy, Precision, Recall, F1, ROC-AUC
- Visualizations: Confusion Matrix, ROC, Precision–Recall
- Streamlit UI for new-customer churn probability prediction
